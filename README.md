# Weather Prism

A modern weather dashboard built with HTML, CSS, and JavaScript.

## Features

- Search weather by city name
- Real-time weather data from OpenWeather API
- Responsive glassmorphism design
- Dynamic weather icons and background based on conditions
- Loading animation while fetching data
- Current date and time display
- Error handling for invalid city names

## Files

- `index.html` — Application structure and UI
- `style.css` — Glassmorphism styling, animations, and responsive layout
- `script.js` — Search handling, API requests, UI updates, and error handling

## Setup

1. Register for a free API key at [OpenWeather](https://openweathermap.org/).
2. Open `script.js`.
3. Replace `YOUR_API_KEY_HERE` with your OpenWeather API key.
4. Open `index.html` in your browser.

## Usage

1. Enter a city name in the search field.
2. Click the `Search` button.
3. View temperature, humidity, weather condition, city name, and live description.

## Notes

- The app uses the OpenWeather current weather endpoint.
- For a better experience, make sure your API key is valid and your internet connection is active.
